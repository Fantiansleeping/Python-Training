# Python format() 方法超详细讲解（新手友好+例题）
## 一、format() 是什么？
`format()` 是 Python 中用于**字符串格式化**的核心方法，简单说就是：  
把变量/数值“填”到字符串的指定位置，让字符串内容更灵活、更易读。  
核心语法：  
```python
"字符串模板".format(参数1, 参数2, ...)
```
- 字符串模板：用 `{}` 作为“占位符”，表示要填充内容的位置；
- format() 里的参数：按顺序填充到 `{}` 中。

## 二、format() 基础用法（必掌握）
### 1. 按位置填充（最常用）
#### 用法1：无编号占位符（按顺序匹配）
```python
# 语法："{}{}".format(值1, 值2)
msg = "我叫{}，今年{}岁".format("小明", 18)
print(msg)
```
#### 运行结果：
```
我叫小明，今年18岁
```
#### 讲解：
- 第一个 `{}` 对应 `format` 里的第一个参数“小明”；
- 第二个 `{}` 对应第二个参数18；
- 不管参数是字符串、数字，都会自动转换成字符串填充。

#### 用法2：带编号占位符（指定顺序）
```python
# 语法："{0}{1}{0}".format(值1, 值2) → 0=第一个参数，1=第二个参数
msg = "我叫{0}，{0}今年{1}岁".format("小红", 20)
print(msg)
```
#### 运行结果：
```
我叫小红，小红今年20岁
```
#### 讲解：
- `{0}` 重复使用第一个参数“小红”，避免重复写参数；
- 编号从0开始，适合需要重复使用同一个参数的场景。

### 2. 按关键字填充（更易读）
#### 用法：占位符写关键字，format 传关键字参数
```python
# 语法："{name}{age}".format(name=值1, age=值2)
msg = "姓名：{name}，年龄：{age}，性别：{gender}".format(
    name="小李", 
    age=22, 
    gender="男"
)
print(msg)
```
#### 运行结果：
```
姓名：小李，年龄：22，性别：男
```
#### 讲解：
- 占位符 `{name}` 对应 `format` 里的 `name="小李"`，和顺序无关；
- 优点：参数多的时候，不用记顺序，看关键字就知道填的是什么，代码更易维护。

### 3. 填充列表/字典（实用扩展）
#### 用法1：填充列表（通过索引）
```python
person = ["小张", 25, "女"]  # 列表
msg = "我是{0[0]}，{0[1]}岁，{0[2]}性".format(person)
print(msg)
```
#### 运行结果：
```
我是小张，25岁，女性
```
#### 讲解：
- `{0[0]}` 表示 `format` 里第一个参数（person列表）的第0个元素（小张）；
- `{0[1]}` 表示person列表的第1个元素（25）。

#### 用法2：填充字典（通过key）
```python
person = {"name": "小王", "age": 19}  # 字典
msg = "名字：{name}，年龄：{age}".format(**person)
print(msg)
```
#### 运行结果：
```
名字：小王，年龄：19
```
#### 讲解：
- `**person` 是“解包字典”，把字典的key-value转换成关键字参数（name="小王", age=19）；
- 占位符 `{name}` 直接匹配字典的key。

## 三、format() 进阶用法（重点）
### 1. 数字格式化（高频考点）
处理数字时，可在占位符里指定格式（比如保留小数、百分比、千分位），语法：  
`{:格式}`  
| 格式 | 含义 | 例子 | 结果 |
|------|------|------|------|
| :.nf | 保留n位小数 | `"{:.2f}".format(3.1415)` | 3.14 |
| :% | 转百分比（乘100加%） | `"{:.1%}".format(0.25)` | 25.0% |
| :, | 千分位分隔符 | `"{:,}".format(1234567)` | 1,234,567 |
| :d | 整数（补0） | `"{:03d}".format(5)` | 005 |

#### 例题：数字格式化实战
```python
# 保留2位小数
print("圆周率保留2位：{:.2f}".format(3.1415926))
# 百分比
print("通过率：{:.1%}".format(0.85))
# 千分位
print("销售额：{:,}元".format(12345678))
# 补0成3位整数
print("编号：{:03d}".format(8))
```
#### 运行结果：
```
圆周率保留2位：3.14
通过率：85.0%
销售额：12,345,678元
编号：008
```

### 2. 字符串对齐（实用）
语法：`{:对齐方式n}`，n是总长度  
- `<`：左对齐（默认）
- `>`：右对齐
- `^`：居中对齐

#### 例题：字符串对齐
```python
# 总长度10，左对齐
print("左对齐：{:<10}结束".format("Python"))
# 总长度10，右对齐
print("右对齐：{:>10}结束".format("Python"))
# 总长度10，居中对齐
print("居中对齐：{:^10}结束".format("Python"))
# 填充字符（比如用-填充）
print("填充：{:-^10}".format("Python"))
```
#### 运行结果：
```
左对齐：Python    结束
右对齐：    Python结束
居中对齐：  Python  结束
填充：--Python--
```

## 四、format() 经典例题（巩固练习）
### 例题1：拼接个人信息
```python
name = input("请输入姓名：")
age = int(input("请输入年龄："))
score = float(input("请输入成绩："))

# 格式化输出
info = "【个人信息】\n姓名：{}\n年龄：{}岁\n成绩：{:.1f}分".format(name, age, score)
print(info)
```
#### 运行示例：
```
请输入姓名：小丽
请输入年龄：17
请输入成绩：98.5
【个人信息】
姓名：小丽
年龄：17岁
成绩：98.5分
```

### 例题2：循环中使用format（结合之前学的for循环）
```python
# 打印1-5的平方，格式对齐
print("数字 | 平方")
print("-----|-----")
for i in range(1, 6):
    print("{:^5}|{:^5}".format(i, i*i))
```
#### 运行结果：
```
数字 | 平方
-----|-----
  1  |  1  
  2  |  4  
  3  |  9  
  4  |  16 
  5  |  25 
```

## 五、总结（核心要点）
1. 基础用法：`{}` 占位符，按位置/关键字填充，是最常用的场景；
2. 数字格式化：`.nf` 保留小数、`%` 转百分比、`, ` 千分位，处理数字必用；
3. 对齐/填充：`<`/`>`/`^` 实现字符串对齐，适合排版输出；
4. 核心优势：比直接拼接字符串（比如 `name + "今年" + str(age) + "岁"`）更简洁、不易错，尤其处理数字时。
